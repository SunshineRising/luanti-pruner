# Pruner Mod for Luanti

Adds a **Steel Pruner** tool that clears connected leaves from trees in one action.

## Features

- **Punch a leaf** — BFS finds and breaks all connected leaves (up to 100)
- **Collects drops** — saplings, sticks, etc. drop as normal
- **Wears down** — ~200 leaf breaks before the tool breaks (~5 full trees)
- **Craftable** — 2 steel ingots + 1 stick in a diagonal shears pattern

## Installation

Clone directly into your Luanti mods directory:

```
cd ~/.minetest/mods
git clone https://github.com/SunshineRising/luanti-pruner.git pruner
```

**Requires:** the default game.
